WeatherView

Proyecto de la materia de Aplicaciones Web Orientadas a Servicios

Este proyecto fue desarrollado para la materia de Aplicaciones Web Orientadas a Servicios, donde implementamos diferentes APIs en una aplicación web. En mi caso, utilicé la API de inicio de sesión de Google, la API del clima OpenWeatherMap y la API de mapas Leaflet. Mi proyecto consiste en una aplicación web de clima en tiempo real, donde los usuarios pueden buscar cualquier ciudad en el mapa para ver su clima actual y también iniciar sesión con Google.
